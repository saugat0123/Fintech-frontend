कागजपत्रहरु

  • आवश्यक जोखिमहरुबाट सुरक्षित गरीएको "बैंकिङ्ग शर्त" (Bankers Clause) सहितको बैंकको पक्षमा धितोबन्धक/दृष्टिबन्धकमा रहेका सम्पूर्ण सामान मौज्दातहरु/सम्पत्तिहरुका लागि पर्याप्त मुल्य बराबरको बैंकको पक्षमा बिमालेख ।
